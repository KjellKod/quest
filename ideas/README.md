# Ideas

This directory tracks future work items that are not yet ready for a full quest.

## What Goes Here

- Feature concepts
- Improvement suggestions
- Integration ideas
- Workflow enhancements

## Format

Each idea is a brief markdown file with:

```markdown
# Idea Title

## What
Brief description of the idea.

## Why
Why this would be useful.

## Approach
Rough implementation ideas (optional).

## Status
idea | in-progress | implemented | rejected
```

## When to Graduate to a Quest

When an idea is ready for implementation, create a quest brief at `.quest/<name>/quest_brief.md` and run `/quest`.
